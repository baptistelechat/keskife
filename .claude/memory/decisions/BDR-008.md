# BDR-008 — Manifest PWA statique dans `public/` (pattern ifecho)

| Décision                                                                          | Pourquoi                                                                                              | Alternatives considérées                                                                     | Statut |
| --------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- | ------ |
| `manifest: false` dans VitePWA + `public/manifest.json` + `<link rel="manifest">` | Manifest visible immédiatement en `pnpm dev` dans DevTools, sans `dev-dist/` parasite ni `devOptions` | `manifest:` dans VitePWA (génère `dev-dist/`) ; `devOptions: {enabled:true}` (même problème) | actif  |
