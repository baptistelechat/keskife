# BDR-012 — `motion/react` retenu pour les animations Keskife

| Décision                                          | Pourquoi                                                                                           | Alternatives considérées                                                        | Statut |
| ------------------------------------------------- | -------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- | ------ |
| `motion` v12 installé, pattern `LazyMotion + m.*` | Exit animations impossibles en CSS pur ; `AnimatePresence` propre pour cards/tab switch/chart swap | CSS `@starting-style` (entrée uniquement, pas de sortie) ; `setTimeout` fragile | actif  |

## Notes

- `LazyMotion features={domAnimation}` placé dans `main.tsx` (root unique — voir GLRN-095)
- Composants : `m.div` partout, jamais `motion.div` (bundle-split)
- Cas d'usage : tab switch (`mode="wait"`), entry cards (`layout` + exit slide-left), chart swap (`mode="wait"`), auth card (scale-in)
- CSS pur conservé pour : `button:active { scale(0.97) }` et `[role="alert"]` slide-in
