# BDR-007 — Keskife est une PWA avec auto-reload silencieux

| Décision                                                                                             | Pourquoi                                                                            | Alternatives considérées                                                                                                     | Statut |
| ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | ------ |
| `vite-plugin-pwa` (`registerType: 'autoUpdate'`) + `controllerchange`+`hadController` dans `App.tsx` | Installable sur mobile/desktop, offline, MAJ transparente sans friction utilisateur | `useRegisterSW`+`onNeedRefresh` (non fiable après 2e cycle SW) ; toast "MAJ disponible" (friction inutile sur SPA stateless) | actif  |

## Notes

- Icône SVG unique (`sizes: "any"`) dans le manifest — Chrome desktop/Android OK, iOS nécessite PNG si homescreen requis
- Voir [LRN-018](../learnings/LRN-018.md) pour le pattern icône SVG
- Le pattern `hadController` capturé avant le `addEventListener` évite un reload au 1er install
