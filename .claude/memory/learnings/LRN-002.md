# LRN-002 — Recharts : couleurs via props, pas via classes Tailwind

| Pattern observé                                                                                | Contexte                                                                                   | Application future                                                                                   |
| ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| Recharts peint les SVG en JS — les classes CSS Tailwind n'atteignent pas les éléments internes | `fill` et `stroke` sur `<Bar>`, `<Pie>`, `<Line>` doivent être des valeurs hex/rgba inline | Toujours passer les couleurs depuis `TAG_COLORS` (défini dans `types/index.ts`) directement en props |
