# LRN-027 — `children` à destructurer de DayButton props pour ajouter du JSX sibling

| Pattern observé                                                                                                                                                                                           | Contexte                                                                                                            | Application future                                                                                                              |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `{...props}` dans `CalendarDayButton` inclut `children` (le numéro de jour). Pour ajouter un élément sibling (ex : dot indicateur), il faut destructurer `children` explicitement et le rendre séparément | Ajout d'un dot `bg-primary/70` sous le numéro de jour — impossible avec `<Button {...props} />` car children écrasé | Toute customisation du contenu `DayButton` qui ajoute du JSX au-delà du numéro de jour nécessite de sortir `children` du spread |
