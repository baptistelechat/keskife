# LRN-020 — shadcn charts : couleurs via `var(--color-{key})` dans les data

| Pattern observé                                                           | Contexte                                    | Application future                                               |
| ------------------------------------------------------------------------- | ------------------------------------------- | ---------------------------------------------------------------- |
| `ChartContainer` injecte `--color-{key}` pour chaque clé du `ChartConfig` | Migration KeskifePieChart + KeskifeBarChart | Toujours passer `fill: "var(--color-{tag})"` dans les data items |

`ChartStyle` génère dynamiquement un `<style>` avec `[data-chart=id] { --color-dev: #3b82f6; ... }`. Il faut donc :

- Pie : `fill: "var(--color-dev)"` dans chaque objet data (+ `nameKey="tag"` sur `ChartLegendContent`)
- Bar : `fill={"var(--color-dev)"}` sur chaque `<Bar>` component
- **Ne pas** passer les hex de `TAG_COLORS` directement dans le JSX chart (l'injection CSS vars est le mécanisme shadcn)
