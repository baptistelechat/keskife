# LRN-018 — Manifest PWA : icône SVG `sizes: "any"` évite la génération PNG

| Pattern observé                                                                                                                                                               | Contexte                                                                   | Application future                                                                                                                      |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Déclarer une icône SVG avec `sizes: "any"` dans le manifest suffit pour Chrome desktop/Android — évite `@vite-pwa/assets-generator` et la génération de PNG multi-résolutions | Keskife PWA setup — seul `icon.svg` dans `public/`, build passe, SW généré | ⚠️ iOS Safari homescreen nécessite PNG `180×180` — SVG seul = icône manquante sur iOS. Ajouter PNG uniquement si support iOS PWA requis |
