# LRN-028 — `getDay()` pour weekends dans rdp v10 (modifier `weekend` absent)

| Pattern observé                                                                                                                                                                  | Contexte                                                                          | Application future                                                                                                 |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| react-day-picker v10 n'expose pas de modifier `weekend` built-in. Utiliser `[0, 6].includes(day.date.getDay())` directement dans `CalendarDayButton` pour styler samedi/dimanche | Tentative de cibler `modifiers.weekend` — propriété absente, aucun style appliqué | En v10, toujours dériver le weekend depuis `day.date` plutôt que compter sur des modifiers built-in non documentés |
